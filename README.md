# 文档自动排版工具

一个基于 Python 的文档自动排版工具，支持 Word 文档和纯文本文档的读取、格式化和排版。

## 功能特性

### 基础功能
- 文件读取：支持 .txt、.md、.docx 格式
- 统一格式设置：默认宋体小四、固定行距 18 磅
- 冗余格式清理：自动清除多余空格、连续空行、无意义换行
- 系统操作菜单：简洁的文本菜单界面
- 错误处理：完善的异常捕获和友好提示
- 退出保存：退出前自动提示保存

### 进阶功能
- 字数统计：支持中英文混合统计
- 自定义格式：可自定义字体、字号、行距、缩进
- 多级标题：支持 Markdown 格式标题

## 技术栈

- Python 3.12
- python-docx 1.1.0（Word 文档处理）
